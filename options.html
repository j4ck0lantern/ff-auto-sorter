Auto Sorter Options

Configure your categories below. The configuration must be a JSON Array (starts with [ ... ]).

The array contains objects, and bookmarks are checked in the order you define them. This gives you full control over priority.

Each object in the array can have three keys:

  • "folder": (Required) A string for the folder path. Use / to create subfolders (e.g., "Programming/Web").
  • "config": (Required) An object containing your matching rules (see below).
  • "index": (Optional) A number or an array of numbers that specifies the folder's position (zero-based).
    • If a number (e.g., 0), it sets the position of the top-level folder.
    • If an array (e.g., [0, 1]), it sets the position of folders in the path ("Folder/Subfolder" would place "Folder" at index 0 and "Subfolder" at index 1 inside "Folder").
    • If omitted, the folder will be moved to the top (index 0) when a new bookmark is added to it.

The "config" object contains your matching rules:

  • "keywords": (Optional) A list of simple string keywords.
  • "comment": (Optional) A string to add notes for yourself.
  • "regex": (Optional) A list of objects, where each object has:
    • "pattern": The regex string. (Remember to escape backslashes: "\\.pdf$").
    • "comment": A string to explain what this regex does.

Example of one item in the array:

[ { "folder": "Programming/Web", "index": [0, 1], "config": { "keywords": ["javascript", "react"], "comment": "Places 'Programming' at the top, and 'Web' at index 1 inside it." } }, { "folder": "Programming/Python", "index": [0, 0], "config": { "keywords": ["python", "django"], "comment": "Places 'Python' at the top of the 'Programming' folder." } }, { "folder": "Utilities", "index": 1, "config": { "keywords": ["converter", "tool"], "comment": "This folder will be locked at index 1 on the toolbar." } }, { "folder": "Gaming", "config": { "keywords": ["gaming", "steam"], "comment": "This folder will move to the top when a bookmark is added." } } ]

A bookmark will be filed under the first rule that matches.
